# Additional Quizzes

## Short Description

The Markdown Quiz Plugin is a powerful, lightweight extension that transforms your markdown documents into interactive quiz experiences. Seamlessly embed questions, multiple-choice options, and instant feedback directly within your markdown files, making it easier than ever to create engaging educational content, documentation, and assessments.
